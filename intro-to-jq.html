Example JS-2A:

Now that we've learned how to CHANGE, ADD and CLEAR content within an element, next we're going to learn how to manipulate the content itself.

  2. Next, it assigned a new variable, var new_string, equal to the first variable, var string
  3. Then, through the JavaScript replace() method, it evaluated the string variable, string.replace, looked for the specified string, string.replace("Students", and replaced it with another string, string.replace("Students", "Your Name");
  4. Our second variable, var new_string, now equaled the first variable, var string, with the word "Students", replaced with the words, "Your Name".

Example JQ-1A:

Similar to the first JavaScript example, in this jQuery example we're going to, again, CHANGE the content of a paragraph element, with a jQuery CLICK event.

Like before, watch the floating container in the top right of your screen to see what happens when you click the button below.

<html>

<head>

<!--Option 1: External URL, Google CDN (Content Distribution Network)-->

<script src = "http://ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js"</script>

<!--Option 2: Internal Path, jquery.js is the file saved in a 'scripts' folder inside the same directory as this file-->

<script src = "scripts/jquery.js"</script>

<!--IMPORTANT NOTE: code only one of these options within the head tag, your aplication will dictate how best to access jQuery-->

</head>

<body>

<div id = "sandbox" style = "color:black;">

<p id = "myFirstParagraph"></p>

<p id = "mySecondParagraph"></p>

</div>

<button id = "example_JQ-1A">See it in Action</button>

<script>

$(document).ready(function() {

$("#example_JQ-1A").on("click", function() {

$("#myFirstParagraph").html("Hey Bob, you can change the color of this text with jQuery!");

});

});

</script>

</body>

</html>

  1. When the browser first hit the <head> tag, it found our <script> tag and followed the src = "url" pointing to Google's CDN and loaded jQuery.
  2. Next, the browser read the rest of the <body> html until it hit another <script> tag that is placed just above the closing </body> tag. Because jQuery was already loaded in the <head> tag, all jQuery methods are define; namely jQuery objects, which are denoted with a dollar sign:  $
  3. The first jQuery method was called on the document, ready() which executed the code within it's function only AFTER the document was ready. Basically after the browser read all the code between the <html> tags.
    NOTE: the ready() method can only be called on the document, which is why we didn't pass a string selector, which we'll explore next.
  4. Even before clicking this <button>, the document was ready and, through jQuery, was listening for an event to be triggered on an element with an id = "example_JQ-1A", which is the selector in this jQuery object, $("#example_JQ-1A")
    NOTE: the hash within the string, #example_JQ-1A, just like CSS selectors, use a # (hash) for an id and a . (period) for a class
  5. The second jQuery method that was triggered from $("#example_JQ-1A") was the on() method, and with it, required two additional pieces: an event and a function separated by a comma .on("event" , function(){ ...code... });
  6. In this example, we used a "click" event and our function() CHANGED the content of our element's tag's html.
    Just like the on() method was called from our button selector, $("#example_JQ-1A"), once the "click" event occured, our target element was found, $("#myFirstParagraph"), and on this a third jQuery method was called, html()
  7. Similar to JavaScript, we found an element by it's id, then passed a string, replacing the html within it's tags. The difference being, in jQuery, the $("stringSelector") keeps us from having to search the document for the element's id, and instead of specifying the innerHTML, html("string") is all you need:
    JavaScript Code:
    document.getElementById("myFirstParagraph").innerHTML = "Hey Bob, you can change the color of this text with jQuery!";
    jQuery Code:
    $("#myFirstParagraph").html("Hey Bob, you can change the color of this text with jQuery!");
  8. Lastly, we needed to close each of the functions and methods with their respective closing syntax. Function code is wrapped in curlly brackets:
    function() { ...code... }
    While method code is wrapped in parenthese:
    method( ...code... )
    Looking at this jQuery function:
    $(document).ready(function() { // first set of opening marks to be closed last
    $("#example_JQ-1A").on("click", function() { // last set of opening marks to be closed first
    $("#myFirstParagraph").html("Hey Bob, you can change the color of this text with jQuery!");
    }); // first set of closing marks
    }); // second set of closing marks
    The first set of }); closes the $("#example_JQ-1A").on("click", function() { ... code ... });
    First close the "click", function() { function }, then the .on( method ) followed with a semi-colon to terminate the line ;
    The second set of }); closes the $(document).ready(function() { ... code ... });
    Close the function() { function }, first, then the .ready( method ) followed with a semi-colon to terminate the line ;

<select id = "example_JQ-1C">
<option value = "black">Black</option>
<option value = "red">Red</option>
<option value = "green">Green</option>
<option value = "blue">Blue</option>
</select>

<script>

$(document).ready(function() {

$().on(, function() {

var selected_color = $(this).val();

$("#sandbox").css("color", selected_color)

});

$("#mySecondParagraph").on(, function() {

var text_color = $("#sandbox").css("color");

console.log("The color of this text is: "+text_color);

});

});

</script>
