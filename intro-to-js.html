LESSON 2: FIND and REPLACE Content

Example 2A:

Now that we've learned how to CHANGE, ADD and CLEAR content within an element, next we're going to learn how to manipulate the content itself.

In this example we are going to FIND and REPLACE a specific word of an element's content and change it to something else.

Watch the floating container in the top right of your screen to see what changes when you click of the button below.

<html>

<head>

<script>

function replace_content() {

var string = document.getElementById("myFirstParagraph").innerHTML;
var new_string = string.replace("Students", "Your Name");
document.getElementById("myFirstParagraph").innerHTML = new_string;

};

</script>

</head>

<body>

<p id = "myFirstParagraph">Hi Students, I am still a paragraph, and my text changed!</p>

<p id = "mySecondParagraph"></p>

<button onclick = "replace_content();">See it in Action</button>

</body>

</html>

When this button was clicked, the in-line onclick event was triggered, which called our function, replace_content();

  1. First, this function took the document and within it, looked for an element with the specified id .getElementById("myFirstParagraph"), and set the content within this element's tag's .innerHTML to the variable, var string
  2. Next, it assigned a new variable, var new_string, equal to the first variable, var string
  3. Then, through the JavaScript replace() method, it evaluated the string variable, string.replace, looked for the specified string, string.replace("Students", and replaced it with another string, string.replace("Students", "Your Name");
  4. Our second variable, var new_string, now equaled the first variable, var string, with the word "Students", replaced with the words, "Your Name".
  5. Lastly, identical to Example 1A, the function took the document and within it, looked for an element with the specified id .getElementById("myFirstParagraph"), took the content within the element's tag's .innerHTML, which originally was "Hi Students, I am still a paragraph and my text changed!" and set it equal to the variable we created, var new_string = "Hi Your Name, I am still a paragraph, and my text changed!"

In this exercise, we're calling a function which will assign two separate variables, in order to find and replace one phrase with another.

Following the order in which variables must be assigned, in the two spaces below, fill in the name of the appropriate variable in each position.

The two variables assigned in our function are: string and new_string.

Example 2B:

When this button was clicked, the in-line onclick event was triggered, which called our function, replace_content_with_paramater('Bob');

  1. First, this function took the document and within it, looked for an element with the specified id .getElementById("myFirstParagraph"), and set the content within this element's tag's .innerHTML to the variable, var string
  2. Next, it assigned a new variable, var new_string, equal to the first variable, var string
  3. Then, through the JavaScript replace() method, it evaluates the string variable, string.replace, looked for the specified string, string.replace("Your Name", and replaced it with another string, string.replace("Your Name", name);
    NOTE: Just like Example 2A, we're replacing text, but this time we are using a variable instead of a string!
    When the function was called in-line, we sent with it, a string. When the function was called, it took that string as a parameter, and assigned it to the variable, name, and because it's a variable and not a string, do not use quotes when using it within the function itself.
    Our second variable, var new_string, now equals the first variable, var string, and with the word "Your Name", replaced with the name parameter we passed in the function.
  4. Lastly, identical to Example 1A, the function took the document and within it, looked for an element with the specified id .getElementById("myFirstParagraph"), took the content within the element's tag's .innerHTML, which originally was "Hi Your Name, I am still a paragraph and my text changed!" and set it equal to the variable we created, var new_string = "Hi Bob, I am still a paragraph, and my text changed!"

In this exercise, we're passing a string through our function, which, when triggered, will take that string, transform it into a parameter and use it within the function.

Example 2C:

For our last example, because we can now pass a string within our function and assign it to a parameter, this time, we are going to pass TWO strings.

Through our function, each will be assigned their own parameter and each will act within the function independently of each other as a result.

Watch the floating container in the top right of your screen when we pass two string with our click event on the button below.

<html>

<head>

<script>

function replace_content_with_two_paramaters(greeting, style) {

var string = document.getElementById("myFirstParagraph").innerHTML;
var first_string = string.replace("Hi", greeting);
var second_string = first_string.replace("text", style);
document.getElementById("myFirstParagraph").innerHTML = second_string;

};

</script>

</head>

<body>

<p id = "myFirstParagraph">Hello Bob, I am still a paragraph, and my color changed!</p>

<p id = "mySecondParagraph"></p>

<button onclick = "replace_content_with_two_paramaters('Hello', 'color');">See it in Action</button>

</body>

</html>

When this button was clicked, the in-line onclick event was triggered, which called the function, replace_content_with_two_paramaters('Hello', 'color');

  1. First, this function took the document and within it, looked for an element with the specified id .getElementById("myFirstParagraph"), and set the content within this element's tag's .innerHTML to the variable, var string
  2. Next, it assigned a new variable, var first_string, equal to the initial variable, string
  3. Then, through the JavaScript replace() method, it evaluates the string variable, string.replace, looked for the specified string, string.replace("Hi", and replaced it with the greeting parameter, string.replace("Hi", greeting);
    This variable, var first_string, was set equal to the initial variable, var string, but with the word "Hi", replaced with the first parameter, greeting, that was passed in the function.
    NOTE: Just like the last Example 2B, we've replaced text, but this time we used the first parameter, greeting
    When the function was called in-line, we sent two string along with it; the function took both strings as parameters, and assigned them to the param variables, greeting and style, respectively.
  4. Now, because we passed a second parameter, style, we used the replace() method again, but this time, it evaluated the first_string variable, string.replace, looked for the specified string, first_string.replace("text", and replaced it with the style parameter, string.replace("text", style);
    Our second variable, var second_string, now equals the first string, var first_string, but with the word "text", replaced with the second parameter we passed in the function, style
    NOTE: When passing more than one parameter through a function, the order in which you use them within the function doesn't matter, however, the order in which you pass them from the in-line function, does: ('Hello', 'color') ... (greeting, style)
  5. Lastly, identical to Example 1A, the function took the document and within it, looked for an element with the specified id .getElementById("myFirstParagraph"), took the content within the element's tag's .innerHTML, which originally was "Hi Bob, I am still a paragraph and my text changed!" and set it equal to the variable we created, var second_string = "Hello Bob, I am still a paragraph, and my color changed!"

In our last exercise, we're passing two strings through our function, and using their respective parameters to replace text within our assigned variables.
